# Assets

Place visual assets here to maximize GitHub visibility:

| File | Recommended size | Purpose |
|------|------------------|---------|
| `logo.png` | 512×512 | README header, package icon |
| `architecture.png` | 1200×630 | Architecture diagram |
| `demo.gif` | ≤ 5 MB | Animated demo for README |
| `social_preview.png` | **1280×640** | GitHub social preview (Settings → General) |

## Social preview setup

1. Add `social_preview.png` (1280×640) to this folder
2. Go to **Repository Settings → General → Social preview**
3. Upload the image

A strong social preview significantly increases click-through from GitHub search and shares.
